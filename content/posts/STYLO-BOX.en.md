This project was developed for the **Digital Systems 2** course at my university. The main task was to build a **jukebox** using an **STM32** microcontroller programmed in **C**.

---

### Extra challenge: custom stylophone

For the optional part, I added a custom **stylophone**. I used sensors for the keys and added features like octave switching. The implementation was complex because the codebase relied on **state machines** and register-level programming without using **HAL**.

---

### Result

Despite the difficulties, the project was a success and I received the **Best Hardware Design** badge.
